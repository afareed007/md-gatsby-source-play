S3 Bucket Resource Module

It is a resource module for s3 bucket creation. This will create a new AWS S3 bucket.

Composite modules instantiate resource modules.

Resoure modules should not be directly used by application templates. Application templates can be created by instantiating composite modules; to upload a static site for example.

Installation

 
npm install @tc/atom-resource-s3-bucket

Inputs
Input Example Description
environment 'dev' tag
teamName 'dev' tag
versioning false whether to version buckets, true by default
loggingTargetBucket 'bucket123' the bucket to push access logs to
website true whether to enable static site hosting

Usage

 
import * as S3Resource from '@tc/atom-resource-s3-bucket';

const mybucket = new S3Resource.S3Resource(
    'bucketName',
    {
        environment,
        teamName,
        versioning=false, //optional, true by default
        loggingTargetBucket: 'test', //optional, blank by default
        website=true, //optional, false by default
    },
    { parent: this }
);
